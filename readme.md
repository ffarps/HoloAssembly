# HoloAssembly
an Augmented Reality Assembly guide App based on my Degree's final project

A unity 3D app for Android, capable of displaying information on an AR overlayed car.

Unity 2022.3.5f1

Vuforia v 10.17

Sounds from https://freesound.org/
